# Lab 01 — OSPF underlay + iBGP-EVPN (full mesh)

Runnable files for this lab: `topology.clab.yml`, `apply/` (per-step snippets),
`configs/` (full per-node configs).

📖 **The complete guide is in the NetForge Labs portal:**
<https://etherhtun.github.io/netforge-labs/labs/lab-01-fullmesh/>

Run it from the repo root:
```bash
./scripts/deploy.sh 01-ospf-ibgp        # boot the fabric
./scripts/apply.sh  01-ospf-ibgp all    # build every step
```
